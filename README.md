# message-boxes

## Prerequisites

| Tool                                                               | Purpose                                                                                   |
| ------------------------------------------------------------------ | ----------------------------------------------------------------------------------------- |
| [NodeJS](https://nodejs.org/en/download/)                          | Runtime                                                       |

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Run tests

```sh
npm run cypress:open
```
